The purpose of this page is to describe the parts of a computer and their function.



The Motherboard

The motherboard is one of the most important parts in a computer, it's where a lot of the other computer parts are mounted also connections for devices like USB connections for phones. Newer more current versions of motherboards hold upgraded parts, motherboards have also been integrated into newer computer systems. Newer models of motherboards are mostly used on PC systems especially among computer gamers which want to make their computers better in order to compete, and be the best.



The CPU

The CPU a.k.a. the central processing unit is piece of the computer which is basically the computer's brain, it can also be called a microprocessor or a processor. The CPU carries out tasks based on the instructions set by the program, it uses simple input/output order with its commands. Over the years CPUs have become smaller and sleeker in order to keep up with computers becoming smaller, and other smaller devices like smartphones and tablets. Before they had CPUs computers had to be rewired in order to perform their tasks. It was not till the late 1940s where the first version of a CPU was made and then it was big servers instead of small computer chips, from there CPUs evolved and became the small chips that we know today.



The RAM

RAM or random access memory, which can be accessed randomly at any point. RAM is the most common form of memory on computers and most devices. RAM is very different from other types of memory and data storage like USB drives or external hard drives. Before there was RAM which was built into computers there was bigger servers and data, and then there was Drum Memory very low tech data could only be taken in the order it was written so if you messed up you had to start all over again. With RAM you could edit and change the data instead of doing the whole thing over again.



The Hard Drive

There are 2 versions of hard drives there are internal hard drives which are held within the computer system, there is also external hard drives which are not inside of any computer system. The internal drive is the main storage unit for the whole computer system it also holds the OS and any other programs. External drives are used more for documents of files that can be stored separately from the main computer, external hard drives can also be used to carry larger data files, pictures documents, etc. An external hard drive can be considered anything from a small USB drive or even a larger one.



The BUS

The BUS is a computer system, that communicates and transfers data, to and from each of the computer's components, the BUS can also be used to transfer information between computers. Since BUS can be between 2 systems it has evolved over the years with bluetooth connections instead of the traditional wires. From there the BUS has not changed except for its speeds which have gone up therefore making computers faster.



The Mouse, Keyboard, and Monitor

A computer is one thing the other parts to it such as the monitor, keyboard and mouse is another different thing. All these other parts of the computer have also changed over the years from components that were built into the computers to components that are just connected with a USB cord. Now monitors are have the whole computer built into the back, with the mouse and keyboard wirelessly connected to the computer.
