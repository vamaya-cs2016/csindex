How to: How I made Images appear and disapear using Javascript?



Step 1: Describing Variables

The first step in creating the Javascript for the car part shopping cart. All variables in a proper javascript need to be defined in order for the code to work properly, and perform as it needs to perform. The variables "part" and "carpart" both variables used to define and identify certian pieces of code that are in the HTML portion of the website. The "carpart" variable defines the selector that the user will use to select the carpart that he/she wants to add to their order, the selector puts out the options the user can choose from. Once the user chooses one the part that they want they simply click on "Add Part" and the part is added to their shopping cart along with a discription of the power it puts out. The id tag "part" is the display where the image of the engine shows up once the user clicks add part. The "Add Part" button is an event listener that only responds to the selector with the id tag "carpart". The "addpart" id refers to the button that is clicked to notify the javascript to add the part that the user has chosen, it is used throughout the javascript. The "totalprice" refers to the total amount that will be added up in parts in the shopping cart. All this was necessary because without the definitions of the variables the code wouldn't work, if there wasn't any definitions then code wouldn't know what the variables mean and javascript wouldn't know what to do.

Step 3: Removing images using Javascript

Making the images disapear with a simple click is simpler than making the images appear since there is no source material needed to make it function. By making a simple event listener that reacts to a click, and making sure that the event listener only removes the "child" the image that has been created. Once it is specified in the code that it is the child image that is the only thing that needs to be removed, removing items from the shopping cart should be fairly simple and easy.
