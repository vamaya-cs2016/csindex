The purpose of this page is to describe and show how to make a webpage in HTML with CSS styling.



Step 1: Establishing the base code

When establishing the base code you are basically entering in what you will build your webpage on. By setting a sort of template to fill in and add code to. This can be completed with the basic code in the image,both start and end lines of HTML establish overall page. The center and body/background tags are to set up a background for the webpage so that it won't be standard white. Both head tags are to place whatever is inside at the topp of the page, for example the navbar. The style tags are also placed in order to set up the style for the whole page like the text color and conctainers, which I will talk about later. In addition the body tags are placed in order to seperate the headers and style from the other content that will go into the page.
