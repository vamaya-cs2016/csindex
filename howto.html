Step 1: Setting up HTML as calculator template

When making the calculator there is 2 sides to it there is the Javascript which makes the calculator function and do what a calculator does and there is the HTML side which sets up the necessary HTML to build the physical features for the calculator. Step 2: Implementing CSS Navbar that connects this page to the rest of the pages

Putting a navbar into the HTML of the calculator is necessary because it gives the user the ability to navigate through not just the to-do list and the how to page but also all the other pages that were created last time like the about me and hardware pages. It links all the pages together, therefore easier for the user to navigate through them. Step 3: Creating the dropdown & inputs for calculator

The inputs for the calculator will be on both sides of the select tags that will be made for the dropdown. The input tags will need seperate ids since they are self closing you will only need the begining tag. Preferably the tags will be serparated by 3 lines of empty code where in the middle there will be the dropdown, also the id the first input should be "num1" and the second should be "num2". When creating the dropdown for the calculator you need to determine what equations you will set up the calculator to perform, I decided on addition, subtraction, division, multiplication, to the power of, and the square root. All simple equations that symbols and names need to go in between an option tag. Before that you will need to set up the select tags with the id selector that will match the variable name that will be put into the javascript for the calculator.



Step 4: Defining the variables, starting Javascript

This part is defining what the the variables are such as + and *. Those symbols are addition and multiplication we know what they mean but the computer only sees and undefined variable that it doesn't know what to do with. So the variable need to be defined in order for the calculator to do the equation and perform. You need to define each of the individual variables, which ever equations you choose to have as options for the calculator to do. For example the division symbol "/". This symbol needs to be defined as division the computer needs to know that when this symbol is selected in the selector that means that the numbers are being divided. That is defining your variables telling the computer what each symbol means and what should happen when it is selected.
